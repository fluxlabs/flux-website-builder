# Project: Flux Website Builder

## Core Value
Rapid-turnaround (48-72 hour) autonomous website generation service for small businesses and entrepreneurs, leveraging AI to build professional Next.js sites from a simple intake form.

## Current Focus
Initial project setup and intake system development.

## Constraints
- **Turnaround:** 48-72 hours from submission to staging link.
- **Reliability:** AI-generated code must be error-free (syntax/Tailwind).
- **Security:** Secure handling of client assets and API tokens.
- **Architecture:** Next.js (App Router), Vercel, Supabase, OpenAI/Anthropic.
