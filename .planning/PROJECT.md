# Project: Flux Website Builder

## Core Value
Rapid-turnaround (48-72 hour) autonomous website generation service for small businesses and entrepreneurs, leveraging AI to build professional Next.js sites from a simple intake form.

## Current Focus
Scaling & Refinement - Enhancing the Admin Dashboard, Onboarding Experience, and Marketing Site.

## Milestone: Scaling & Refinement
The goal of this milestone is to move from a "MVP" state to a "Growth-Ready" state by refining the core user journey and adding necessary administrative visibility.

### Objectives
1.  **Admin Visibility:** Implement system logs and client management tools.
2.  **Conversion Optimization:** Overhaul the onboarding flow into a premium wizard experience.
3.  **Social Proof & Trust:** Expand the marketing site with detailed pricing and testimonials.

## Constraints
- **Turnaround:** 48-72 hours from submission to staging link.
- **Reliability:** AI-generated code must be error-free (syntax/Tailwind).
- **Security:** Secure handling of client assets and API tokens.
- **Architecture:** Next.js (App Router), Vercel, Supabase, OpenAI/Anthropic.
